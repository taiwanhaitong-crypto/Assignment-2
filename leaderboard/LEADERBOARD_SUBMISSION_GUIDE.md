# Leaderboard Submission Guide

If the course uses a leaderboard for trajectory accuracy:

1. Export your best trajectory (e.g. `CameraTrajectory.txt` or `KeyFrameTrajectory.txt`) in TUM format.
2. Fill in `submission_template.json` with your student ID and run metadata.
3. Submit according to course instructions.

Evaluation metrics (if using evo): ATE RMSE, RPE drift, completeness. See main README for methodology.
